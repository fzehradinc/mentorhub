## 📐 Component Dimensions

### Badge Cards
- **Mobile**: 160×120px
- **Desktop**: 200×150px
- **Icon**: 32×32px
- **Border Radius**: 16px
- **Padding**: 16px

### Upload Zones
- **Document Upload**: 320×200px
- **Logo Upload**: 150×150px
- **Selfie Capture**: 200×200px
- **Drag Zone**: 2px dashed border

### Form Elements
- **Input Height**: 44px minimum
- **Button Height**: 44px minimum
- **Modal Width**: 400px (mobile), 600px (desktop)
- **Section Spacing**: 24px between sections

## 🎯 Interaction Patterns

### Badge Selection
- **Tap/Click**: Show badge details modal
- **Available**: Highlight with blue border
- **Earned**: Show with checkmark overlay
- **Locked**: Show requirements tooltip

### Document Upload
- **Drag & Drop**: Visual feedback on hover
- **File Selection**: Native file picker
- **Progress**: Real-time upload progress
- **Preview**: Thumbnail after upload

### Form Validation
- **Real-time**: Validate on blur
- **Error States**: Red border + error message
- **Success States**: Green border + checkmark
- **Helper Text**: Gray text below inputs
