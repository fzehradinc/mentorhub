## 📐 Component Dimensions

### Status Toggle
- **Mobile**: Full width, 44px height
- **Desktop**: 300×44px
- **States**: 3 radio buttons
- **Animation**: Smooth transition

### SEO Input Fields
- **Title**: 60 character limit
- **Description**: 160 character limit
- **Slug**: 3-60 characters, kebab-case
- **Keywords**: Tag input, max 10

### Analytics Cards
- **Mobile**: Full width, stacked
- **Desktop**: 3-column grid
- **Card Size**: 200×150px
- **Status**: Active/Inactive indicators

**Export Formats**: PNG, SVG, Figma Components
